# UofT Foods

## Description
A Flutter Mobile App designed to aid UofT Students and Staff easily find food options across all campuses.

## Features
- Campus Selection Filters
- Store Tags
- Store hours
- Open notifier
- Search

### Coming soon
- Better Filtering
- Sorting
- "Nearby"
- "Navigate"
- Ratings
- Favorites
